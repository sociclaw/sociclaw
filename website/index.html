Questions?

Do I need auto-posting?

No. SociClaw delivers ready drafts into Trello/Notion. Publish manually or automate later.

What do I need to configure?

API keys (X, Trello/Notion, SociClaw image API). Credits and billing run through SociClaw.

Can I change niche?

Yes. Run /sociclaw setup again and generate a new quarterly plan.
